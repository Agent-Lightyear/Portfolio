# Saad Sheikh Portfolio

A responsive personal portfolio website built with HTML, CSS, and JavaScript to showcase skills, education, internship experience, and featured projects.

## Overview

This portfolio includes:

- A modern landing page with hero, about, skills, projects, and contact sections
- Responsive design for desktop, tablet, and mobile devices
- A hamburger navigation system for smaller screens
- Interactive project cards with modal previews
- Animated skills and typing effects
- JavaScript-based contact form validation

## Featured Projects

- Elite Autohaus
- Forza Cars
- Student-Teacher Appointment Booking
- Upcoming cybersecurity web application

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Firebase

## Project Structure

- `index.html` - main portfolio page
- `index3.html` - alternate portfolio page version
- `style4.css` - main stylesheet for the current layout
- `script3.js` - main JavaScript for interactions and animations
- `resume.pdf` - downloadable resume
- Project images and other assets used throughout the site

## Features

- Sticky responsive navbar
- Mobile and tablet hamburger menu
- Smooth scrolling navigation
- Active section link highlighting
- Typing text animation
- Scroll reveal effects
- Animated progress bars and circular skill meters
- Project modal with live links
- Back to top button
- Contact form validation

## Getting Started

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No build tools or package installation are required.

## Customization

- Update your profile and about details in `index.html`
- Replace project screenshots with your own images
- Add your real social media links
- Keep `resume.pdf` updated with your latest resume
